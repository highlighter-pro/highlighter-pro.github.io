# Highlighter.pro 

**[Highlighter.pro](https://chromewebstore.google.com/detail/highlighterpro/gpdhplhmppgenpnkfilghnjiodfmnoap)** — это расширение Chrome для выделения текста на веб-страницах и (опционально) добавления примечаний к отмеченному фрагменту. Он работает как маркер для веб-страниц, подсвечивая отметку цветом.

Оно *очень* быстрое, очень легковесное и не перегружает ваш браузер тяжелыми скриптами.

У него открытый исходный код.
Вы можете увидеть его код на [GitHub](https://github.com/highlighter-pro/highlighter) и [GitLab](https://gitlab.com/highlighter-pro/highlighter).

Вы можете установить его из репозитория кода, а также из [интернет-магазина Chrome](https://chromewebstore.google.com/detail/highlighterpro/gpdhplhmppgenpnkfilghnjiodfmnoap).

Он не передает никакие ваши данные на удаленный сервер и не требует разрешения на отправку/получение данных в Интернет.
